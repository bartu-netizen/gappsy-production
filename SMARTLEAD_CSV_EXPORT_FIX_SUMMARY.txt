HOW IT WORKS NOW
================================================================================

1. Browser preflight:
   - Sends OPTIONS request with x-admin-token header
   - Edge function responds with CORS_HEADERS (includes x-admin-token)
   - Browser preflight succeeds

2. Actual request:
   - Browser sends GET request with x-admin-token
   - Edge function authenticates and processes
   - Returns CSV with CORS_HEADERS in response

3. Download:
   - Browser receives blob
   - Creates object URL
   - Triggers download
   - Waits 5 seconds
   - Revokes URL

Result: Reliable CSV download, no "Failed to fetch"

  - Actual request never sent
