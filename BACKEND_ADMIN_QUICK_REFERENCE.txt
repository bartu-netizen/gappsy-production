Upload:

2. Upload CSV (Tab tab)
3. Select State: New Jersey
4. Upload

Result:
- 2 agencies created with is_active = true
- All scraper fields stored in database
- Visible in Manage Agencies → Edit

Example 2: Filter and Edit

Manage Agencies:
1. Status filter: "Active" (default)
2. See active agencies
3. Click edit icon on any agency
4. Scroll to "OPTIONAL SCRAPER DATA"
5. Edit zip_code, facebook_url, etc.
6. Click Save

Result:
- Changes saved to database
- List refreshes
- Scraper data persisted

Example 3: Reactivate Deactivated Agency

Manage Agencies:
1. Status filter: "Inactive"
2. See deactivated agencies
3. Click edit icon
4. Check "Active" checkbox
5. Click Save

Result:
- Agency reactivated (is_active = true)
- Moves to Active list
- Scraper data preserved
