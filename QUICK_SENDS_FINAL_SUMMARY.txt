SUPPORT & TROUBLESHOOTING
================================================================================

If "Failed to Fetch" Still Shows:
  1. Add ?debug=1 to URL
  2. Try sending
  3. Check debug panel status
  4. See "Status Code Reference" in test guide

If Email Doesn't Send:
  1. Verify SMTP configured (/wp-admin/email/settings)
  2. Verify email format is valid
  3. Check debug panel for error
  4. Check Supabase edge function logs

If Can't Access Page:
  1. Verify you're logged in as admin
  2. Verify sidebar shows
  3. Verify other admin pages work
  4. Check browser console for errors

Debug Mode:
  Add ?debug=1 to URL to see:
  - Request URL and method
  - Response status code
  - Response body (JSON)
  - Full error details
