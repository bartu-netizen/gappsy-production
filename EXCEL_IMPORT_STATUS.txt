================================================================================
KEY FEATURES
================================================================================

✅ Auto-Column Mapping
   Maps scraper columns (case-insensitive) to agency fields
   - Name, Website, Phone, Email, City, Zip, Description, Services
   - Unknown columns safely ignored

✅ Data Validation
   - City: 2-50 chars, rejects state abbrevs/digits-only
   - Zip: 5-digit or 9-digit ZIP format
   - Auto-trimmed whitespace

✅ Multi-Email Support
   - Multiple emails in one cell supported (comma/semicolon separated)
   - Backend handles deduplication
   - Same behavior as CSV imports

✅ Client-Side Processing
   - No backend Excel dependencies
   - Fast (typically <1 second for 100+ rows)
   - Clear error messages

✅ Backward Compatible
   - CSV imports unchanged
   - JSON imports unchanged
   - /your-agency funnel unaffected
   - No database migrations needed
