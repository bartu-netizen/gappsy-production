## RESPONSIVE BEHAVIOR

### Desktop (1024px+)
```
✓ Stat cards: Side by side (2 columns)
✓ Timer: Full width below stat cards
✓ Clock icon: 16px, left aligned
✓ Text: Single line or wraps naturally
✓ All spacing: Comfortable (px-2.5, py-2)
```

### Tablet (768px - 1023px)
```
✓ Stat cards: Side by side (2 columns, slightly narrower)
✓ Timer: Full width below stat cards
✓ Clock icon: 16px, left aligned
✓ Text: May wrap to 2-3 lines
✓ Spacing: Same as desktop
```

### Mobile (< 768px)
```
✓ Stat cards: Grid layout (2 columns, compact)
✓ Timer: Full width below stat cards
✓ Clock icon: 16px, left aligned
✓ Text: Wraps to 3-4 lines
✓ Font size: Slightly smaller (text-xs)
```
