================================================================================
DETERMINISTIC SERVICE SELECTION
================================================================================

How it works:

1. Uses agency SLUG as seed
   slug = "acme-web-design-new-jersey"

2. Creates hash from slug
   hash = simpleHash(slug) = 12345

3. Selects starting position in service pool
   startIndex = hash % 14 = 7

4. Selects 4 or 5 services
   count = hash % 2 == 0 ? 4 : 5

5. Result: Services starting at index 7 + count
   Services: [Performance Marketing, UX/UI Design, Web Development, Analytics]

Benefits:
✅ Same agency always gets same services
✅ Different agencies get varied services
✅ No randomness on refresh
✅ Re-import preserves existing services

VIEW ACTION
================================================================================

How to use:

1. Go to Admin → Other Agencies Import
2. Click "Manage Agencies" tab
3. Click eye icon on any agency
4. Modal opens showing:
   - Name
   - State
   - City (if available)
   - Website URL (if available, clickable)
   - Phone (if available)
   - Services (as blue badges)
   - Active/Inactive status

5. Click Close to close modal

DELETE ACTION (SOFT DELETE)
================================================================================

How to use:

1. Go to Admin → Other Agencies Import
2. Click "Manage Agencies" tab
3. Click trash icon on any agency
4. Confirmation modal appears
5. Confirm dialog says: "This will deactivate the agency. It can be reactivated later."
6. Click Delete button
7. Agency is DEACTIVATED (not deleted)
8. Agency disappears from active list
9. Agency still in database (recoverable via backend)

Important:
✅ This is SOFT DELETE (sets is_active = false)
✅ Not hard delete (data not removed)
✅ Reversible if needed
✅ Agency removed from Manage list immediately

================================================================================
BACKEND ENDPOINTS
================================================================================

GET /other-agencies-admin/{id}
- Purpose: Fetch single agency details
- Auth: No auth required (anyone can read)
- Response: { ok: true, agency: {...} }

PUT /other-agencies-admin/{id}
- Purpose: Update agency details
- Auth: Required - x-admin-token header
- Body: { name, website_url, phone, services, is_active }
- Response: { ok: true, agency: {...} }

DELETE /other-agencies-admin/{id}
- Purpose: Deactivate agency
- Auth: Required - x-admin-token header
- Response: { ok: true, agency: {...} }
